TIDBF
=====

The Infinite Dubstep Baby Factory

Dependencies
------------

You'll need node.js installed:

```shell
http://nodejs.org/
```

And Grunt's command line interface:

```shell
https://github.com/gruntjs/grunt-cli
```

Getting Started
---------------

Download node dependencies using:
```shell
npm install
```

Run the server using:
```shell
grunt
```

Lint the JS with:
```shell
grunt jshint
```
